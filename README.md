## Setup

To run this project locally, follow these steps:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/username/portfolio.git
2. Install the required dependencies:
   ```bash
   npm install
3. Start the local server:
   ```bash
   npm start
