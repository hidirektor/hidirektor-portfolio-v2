GelKurye - Mobile App

Client: Gel Kurye
Category: Logistic
Date: 01.11.2025
Website: https://github.com/hidirektor/gelkurye-mobile-android

This project was specially developed for GelKurye, a company focused on modernizing and streamlining the motorcycle courier (moto courier) industry. The main goal of the project is to digitize operational workflows, increase efficiency, and provide a user-friendly digital platform for both couriers and clients.

Through this system, couriers can receive, manage, and complete delivery tasks seamlessly using a mobile application, while administrators and dispatchers can monitor operations in real time via the backend dashboard. The platform consists of a Java-based backend powered by Spring Boot and Hibernate for data management, with MySQL as the primary database. On the client side, a native iOS application was developed using Swift, ensuring fast and responsive user experience tailored for Apple devices.
